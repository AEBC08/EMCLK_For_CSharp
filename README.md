### 引用本项目示例
**本项目支持被直接引用并调用函数启动 Minecraft  
以下是调用示例：**

```C#
using IMCLK;

Kernrl.LaunchMinecraft("Your Java path", "Your .minecraft dir path", "Minecraft version name", 1024, "Player name", OutJvmParams: true);
```
